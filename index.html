## Featured Tutorial
📚
Open-Source Course

R Classification: Drone Image Processing

A hands-on, 11-step tutorial on processing drone-captured imagery using R for soil removal, ML classification, and vegetation index extraction. Covers Random Forest, SVM, and XGBoost for multispectral image analysis in agricultural settings.

R 4.4.3 Random Forest SVM XGBoost Multispectral Pix4D / Metashape MIT License

What You Will Learn

  1. Import multispectral bands from drone flights
  2. Create soil and plant training samples
  3. Train ML models (RF, SVM, XGBoost) for classification
  4. Generate classified maps and evaluate model metrics
  5. Remove soil background from imagery
  6. Calculate vegetation indices with pliman & FieldImageR
  7. Extract mean values and export to Excel
